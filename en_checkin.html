Entering the Building

The main entrance is equipped with a digital door lock.

  • Booking.com / Airbnb Reservations
    Please enter the last 4 digits of the password you registered when booking.
  • Reservations from Other Platforms
    The host will provide the door lock password separately via message.

If the door lock password requires an additional key, please press the * key after entering the numbers.
Make sure to enter the digits + * to unlock.
